​	 **项目由学生端，教师端和管理员三个部分组成**

## 管理员页面

```
/adminLogin
```

## 项目启动

- 文件路径为 bin/gdesion

### Compiles and hot-reloads for development

```
npm run dev
```

## 技术栈

### 前端

- vue+vue Router
- element-ui
- moment
- axios

### 后端

- PHP
- MySQL

## 注意项

- 请将整个项目放到php环境下，防止图片加载出现问题。
- 记得将php文件的连接数据库的信息改成自己数据库的信息。
